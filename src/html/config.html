Connection

The app automatically detects your Protect version by reading the version label in the sidebar (e.g. "Protect 7.1.0"). If detection fails, the fallback selected above is used. Legacy (3.x) uses a different UI handler. Modern (4.x and above, including 7.x+) all share the same UI structure and are handled identically – future versions are supported automatically.

Multi-profile configuration

Manage multiple NVR profiles or dashboards. Each profile stores a separate URL and credentials. Use F10 during liveview to cycle through profiles, or right-click the tray icon to switch directly.

Supported Versions

Protect 2.x Protect 3.x Protect 4.x Protect 5.x+

How to get your Liveview URL

Open Unifi Protect in your browser, navigate to your liveview and copy the URL from the address bar.

Profile Switching

Add multiple profiles for different NVRs or dashboards. Press F10 in liveview to cycle through them, or pick one from the system tray menu.
