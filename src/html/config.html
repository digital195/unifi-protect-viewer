Startup Profile

Display & Fullscreen
The list reflects currently connected monitors. Connect all desired displays before selecting.

Startup Behaviour

These settings are global and apply on every launch, independent of the active profile.

Startup Profile: Choose which NVR profile to open automatically. When set to "None", the profile selection screen is shown so you can pick manually.

Fullscreen: Moves the window to the selected monitor and enters fullscreen mode immediately on launch. Default is off.

Multi-Monitor Tips

Make sure all monitors are connected before launching the app. You can also toggle fullscreen manually at any time with F11.

Display numbering follows the order reported by your operating system. The Primary display is your main desktop monitor.
