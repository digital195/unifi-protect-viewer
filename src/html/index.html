About

Unifi Protect Viewer is an open-source desktop wrapper for Unifi Protect.

Click a profile to connect. Use F10 during liveview to cycle between profiles.

Keyboard Shortcuts

F9Restart
F10Edit Configuration
F11Toggle Fullscreen

Could not connect

The liveview URL could not be reached. This is usually caused by a wrong URL, incorrect credentials, or the Unifi Protect instance being unavailable.

Use the Edit Configuration button on the left to update your settings.

Make sure your URL looks like one of these:

https://192.168.1.1/protect/dashboard/<id>

https://192.168.1.1/protect/liveview/<id>
